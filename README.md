# KrishiPrabandh - Agricultural Management System

A clean, professional full-stack architecture for agricultural logistics and field operations.

## Project Structure

```
/frontend    - Vite-powered UI with TailWind CSS
/backend     - Express.js server and Python utility scripts
```

## Getting Started

### Frontend (UI)
1. Navigate to `/frontend`: `cd frontend`
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`

### Backend (API & Utils)
1. Navigate to `/backend`: `cd backend`
2. Install dependencies: `npm install`
3. Run server: `npm start`
4. Python Utilities: Found in `backend/utils/`. Run them using `python backend/utils/<script_name>.py`.

## Features
- **Smart GR Parser**: AI-powered governance decision extraction.
- **Beneficiary Radar**: Proactive inclusion of eligible farmers.
- **Geo-Verified Ops**: Field verification with GPS and camera integration.
- **Policy Friction Analytics**: Identifying bottlenecks in scheme delivery.
