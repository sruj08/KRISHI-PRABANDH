# KrishiPrabandh Backend

This is the Express.js backend and utility collection for KrishiPrabandh.

## Directory Structure
- `/src`: Express.js server logic
- `/routes`: API route definitions
- `/controllers`: Business logic
- `/models`: Database schemas
- `/utils`: Python utility scripts for data processing and layout fixes.

## Getting Started
```bash
npm install
npm start
```

## Python Utilities
- `rewrite.py`: Merges UI components into the main dashboard.
- `apply_changes.py`: Applies translations and layout fixes to HTML files.
