# KrishiPrabandh Frontend

This is a Vite-powered frontend for the KrishiPrabandh application.

## Tech Stack
- **Framework**: Vite
- **Styling**: Tailwind CSS & Vanilla CSS (`src/styles/style.css`)
- **Icons**: Material Symbols Outlined & FontAwesome
- **Maps**: Leaflet.js
- **Charts**: Chart.js

## Directory Structure
- `/src/components`: UI components
- `/src/pages`: Application pages (currently HTML files in root of frontend)
- `/src/styles`: CSS files
- `/src/assets`: Images and static assets
- `/public`: Public assets served directly

## Development
```bash
npm install
npm run dev
```
